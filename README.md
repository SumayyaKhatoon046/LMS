# 🚀 AI-Powered Learning Management System (LMS)

An industry-level AI-powered Learning Management System built using the MERN Stack. This platform enables students to learn courses, track progress, submit assignments, earn certificates, and leverage AI-powered career and interview assistance.

---

## ✨ Features

### 👨‍🎓 Student Features

* User Authentication (JWT)
* Browse Courses
* Enroll in Courses
* Watch Lectures
* Track Learning Progress
* Submit Assignments
* Download Certificates
* Personal Dashboard
* Profile Management

### 👨‍🏫 Teacher Features

* Create Courses
* Manage Lectures
* Upload Resources
* Review Assignments
* Track Student Progress

### 👨‍💼 Admin Features

* Manage Users
* Manage Courses
* Dashboard Analytics

### 🤖 AI Features

* AI Career Predictor
* AI Roadmap Generator
* AI Interview Questions
* Resume Review
* Resume Score
* Mock Interview
* Communication Coach
* AI Coding Hints
* Placement Readiness
* Study Planner
* Revision Reminder
* AI Code Review *(Gemini Integration Ready)*

## ⚙️ Installation

### Clone Repository

```bash
git clone <YOUR_GITHUB_REPOSITORY_URL>
```

### Backend

```bash
cd backend
npm install
npm run dev
```

### Frontend

```bash
cd frontend
npm install
npm run dev
```

---

## 🔑 Environment Variables

Create a `.env` file inside the backend folder.

```env
PORT=5000
MONGO_URI=YOUR_MONGODB_URI
JWT_SECRET=YOUR_SECRET
CLOUDINARY_CLOUD_NAME=YOUR_CLOUD_NAME
CLOUDINARY_API_KEY=YOUR_API_KEY
CLOUDINARY_API_SECRET=YOUR_API_SECRET
GEMINI_API_KEY=YOUR_GEMINI_API_KEY
EMAIL=YOUR_EMAIL
EMAIL_PASSWORD=YOUR_APP_PASSWORD
```
